How to Compile:
This project contains 3 folders: Feature Extraction, Model Building, and Result. 

1. mlproj_features.py
This file contains code for feature extraction. To run it, type python mlproj_features.py in the terminal. After execution, copy and replace the train and test files generated to the Model Building folder.

2. mlproj_models.py
This file contains code for model building and outcome prediction. To run it, type python mlproj_models.py in the terminal.
